# Iris Flower Classification 🌸

This project is part of the Oasis Infobyte Data Science Internship.

## Objective
To classify iris flowers into Setosa, Versicolor, and Virginica using Machine Learning.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Model
Random Forest Classifier

## Result
The model achieved high accuracy and correctly predicts flower species.
